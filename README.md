## 🎯 Как изменить контакты

### Telegram
В `index.html` найдите (строка ~261):
```html
<a href="https://t.me/your_username">
```
Замените на:
```html
<a href="https://t.me/ваше_имя_в_телеграм">
```

### Email
Найдите (строка ~264):
```html
<a href="mailto:your.email@example.com">
```
Замените на:
```html
<a href="mailto:ваша.почта@gmail.com">
```

## 🎬 Как добавить свои видео

### Для каждой буквы в `data.js`:

```javascript
{ 
    letter: 'ا',           // Буква
    name: 'Алиф',          // Название
    transcription: 'A',    // Произношение
    examples: 'أم (мать)', // Примеры
    videoUrl: 'https://www.youtube.com/embed/K8D6jm4FfPs'  // ← ЗАМЕНИТЕ
}
```

### Как получить ссылку на видео:

1. Откройте видео на YouTube
2. Скопируйте ID из URL:
   ```
   https://youtube.com/watch?v=K8D6jm4FfPs
                                 ↑ это ID видео
   ```
3. Вставьте в ссылку:
   ```
   https://www.youtube.com/embed/K8D6jm4FfPs
   ```

## 🐛 Возможные проблемы

### Видео не загружается
- Проверьте интернет
- Попробуйте другой браузер
- Убедитесь что videoUrl правильный

### Прогресс не сохраняется
- Проверьте что cookies/localStorage не отключены
- Удалите кэш браузера
- Попробуйте приватный режим (InPrivate)

### Приложение не загружается
- Убедитесь что все файлы в одной папке
- Проверьте консоль браузера (F12)
- Попробуйте открыть через `http://` (не просто файл)
