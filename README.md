
# Платформа для изучения арабского языка

**Курсовой проект**  
*Выполнил: Абдуллоев О.О., группа ПИЖ-б-о-25-1(1)*

## О проекте
Интерактивная веб-платформа для изучения арабского языка. Включает изучение алфавита, словарь по темам, тесты и отслеживание прогресса.

## Функционал
- **Главная страница** — информация об арабском языке + случайные пословицы
- **Алфавит** — 28 букв с транскрипцией, примерами и модальным окном
- **Словарь** — тематические разделы (приветствия, числа, семья, цвета, еда)
- **Тесты** — проверка знаний алфавита и слов
- **Прогресс** — статистика обучения (сохранение в localStorage)
- Тёмная тема
- Адаптивный дизайн

## Технологии
- HTML5
- CSS3 (включая тёмную тему)
- JavaScript (vanilla)
- Google Fonts (Amiri, Inter, Playfair Display)
- Font Awesome

## Как запустить
1. Распакуйте архив
2. Откройте файл `index.html` в любом современном браузере

Сайт полностью работает **локально** без сервера.
