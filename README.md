
# Платформа для изучения иностранного языка 

Курсовой проект по дисциплине «Информационные технологии и программирование».
Разработка клиентского веб-интерфейса для изучения иностранного языка: алфавит, тематический словарь, тесты на проверку знаний и отслеживание прогресса обучения.

Репозиторий:
https://github.com/Abdulloevo7/my-site

---

# Техническое задание

## Общие требования

Назначение программного продукта — создание интерактивной веб-платформы для изучения арабского языка. Сайт даёт пользователю возможность изучать арабский алфавит, пополнять словарный запас по тематическим разделам, проверять знания с помощью тестов и следить за собственным прогрессом обучения.

Проект реализован как клиентское веб-приложение (frontend-only) без серверной части. Все данные хранятся локально в коде приложения и в localStorage браузера.

---

## Функциональные требования

### Обязательные:
- главная страница с информацией об арабском языке;
- раздел «Алфавит» — 28 букв с транскрипцией, примерами слов и модальным окном;
- раздел «Словарь» с тематическими разделами;
- раздел «Тесты» — проверка знаний алфавита и слов;
- раздел «Прогресс» — статистика обучения;
- адаптивный интерфейс;
- переключение светлой и тёмной темы;
- сохранение данных в localStorage.

### Желательные:
- модальные окна с дополнительными примерами и видеоматериалом по буквам;
- слайдер с арабскими пословицами на главной странице;
- hover-эффекты и плавные анимации;
- динамическое обновление статистики и прогресс-баров.

---

## Технические требования

Сайт функционирует на современных компьютерах и мобильных устройствах. Минимальные технические характеристики: процессор от 1,6 ГГц, ОЗУ от 512 МБ. Рекомендуемое разрешение экрана — от 1024×768 пикселей.

---

## Программные требования

Сайт работает в современных браузерах:
- Google Chrome;
- Mozilla Firefox;
- Microsoft Edge;
- Safari.

Разработка выполнялась в Visual Studio Code с использованием Git и GitHub.

# Запуск проекта

## Локальный запуск

1. Клонировать репозиторий:

git clone https://github.com/Abdulloevo7/my-site.git

2. Открыть проект в Visual Studio Code.

3. Запустить проект через расширение Live Server
или открыть файл index.html в браузере.

Сайт полностью работает локально, без сервера.
